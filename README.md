### Controls
* `A, D` - move left/right
* `Left Shift` - run
* `Space` - jump
* `Q` (hold) - lantern fuel exchange
* `E`, `Secondary Mouse Button` - toggle lantern on/off
* `Primary Mouse Button` - attack
